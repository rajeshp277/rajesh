# 🚀 Featured Machine Learning & Geospatial Projects:

* **Self-Supervised Deep Learning for Semantic Segmentation (Master's Thesis):** Fine-tuned a pretrained DINOv2 Vision Transformer (ViT-S/14) utilizing Supervised Contrastive Learning (SupCon) in PyTorch to classify structural phases in complex microscopic imagery, achieving 93.76% accuracy.
* **Single Image Tracking via Siamese Fully Convolutional Networks:** Designed and optimized a Siamese FC Network in PyTorch for high-precision single object tracking, integrated seamlessly with OpenCV for real-time video analysis.
* **Machine Learning for Low-Cost LiDAR (VLP-16) Quality Optimization:** Extracted spatial features from Velodyne (VLP-16) scanners and laser trackers, training machine learning regression models to significantly enhance 3D measurement precision.
* **Bathymetric Data Simulation & Uncertainty Analysis:** Analyzed shallow water geospatial data from Multibeam Echo sounders and LiDAR. Utilized Helios++ and applied Monte Carlo Simulations to drastically improve spatial model accuracy.
* **Automated SAR Processing & Flood Mapping:** Developed scalable Python pipelines (GeoPandas, GDAL, Rasterio) to process multi-temporal Sentinel-1 SAR data, producing GIS-ready flood extent layers and ensuring ISO-compliant metadata.
